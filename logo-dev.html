morphlogo: demo

morphlogok ist ein animiertes Logo für Webseiten.

Die Einbindung ist denkbar einfach: Der javascript-Code wird mit

<script type='text/javascript' language='javascript' src='morphlogo.min.js'></script>
nach dem body-Tag eingefügt, das Logo selber wird mit
<div id="morphlogo"></div>
eingesetzt und aktiviert.

morphclock is an animated logo for web pages.

Embedding is quite easy: insert the javascript code after the body-tag like this:

<script type='text/javascript' language='javascript' src='morphlogo.min.js'></script>
and activate the logo by inserting
<div id="morphlogo"></div>
anywhere on the web page.
